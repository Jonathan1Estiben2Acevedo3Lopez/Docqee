# Design System Documentation

## Theme Overview

This document outlines the core design decisions that shape the user experience.

## Colors

The design employs a 'light' color mode.
* **Primary Color:** `#00647c` - A vibrant hue, ideal for calls to action and prominent interactive elements.
* **Secondary Color:** `#64748B` - A sophisticated supporting color, suitable for less emphasized UI components.
* **Tertiary Color:** `#F0F9FF` - A soft accent color, used for highlights and decorative touches.
* **Neutral Color:** `#F8FAFC` - A crisp neutral base, perfect for backgrounds and foundational surfaces.

## Typography

The chosen typefaces contribute to the system's aesthetic and readability:
* **Headlines:** `manrope` - Provides a modern and impactful presence.
* **Body Text:** `publicSans` - Ensures clear and accessible reading for paragraphs and longer content.
* **Labels:** `publicSans` - Offers consistent readability for interactive labels and small text.

## Shape & Form

* **Roundedness:** `2` (Moderate) - UI elements feature a moderate level of rounded corners, striking a balance between modern softness and clear definition.

## Spacing

* **Spacing:** `2` (Normal) - The layout utilizes a normal level of whitespace, providing adequate breathing room between elements without feeling overly sparse or dense.
